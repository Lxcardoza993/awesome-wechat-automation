# Design: Awesome WeChat Automation

## Goal

Create a curated, high-quality Awesome List repository focused primarily on **WeChat Official Account (微信公众号) auto-publishing and operation automation**, while including a dedicated "Related WeChat Ecosystem Automation" section for high-value adjacent tools (mini-programs, Enterprise WeChat, WeChat bots, etc.).

## Scope

### In scope (core)

- Tools that automate publishing articles to WeChat Official Accounts.
- Article drafting / AI writing assistants for WeChat articles.
- Media asset management for WeChat (images, videos, drafts).
- Analytics and data monitoring for WeChat Official Accounts.
- Multi-platform-to-WeChat publishing workflows (e.g., Notion/Markdown → WeChat).
- SDKs, API wrappers, and utilities for WeChat Official Accounts.

### In scope (related / extended)

- WeChat Mini Program automation tools.
- Enterprise WeChat (WeCom) automation tools.
- WeChat personal account bots / customer service automation.
- Browser automation / RPA specifically targeting WeChat web interfaces.

### Out of scope

- Generic social media automation tools with no WeChat-specific support.
- WeChat Pay-only SDKs unless related to broader automation.
- Closed-source SaaS without a clear WeChat automation feature.

## Proposed Repository

- **Repository name:** `Lxcardoza993/awesome-wechat-automation`
- **Visibility:** public
- **License:** CC0-1.0 (content), with individual tools retaining their own licenses.
