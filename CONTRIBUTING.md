# Contributing / 贡献

Contributions are welcome! When proposing a new tool, please provide:

- Tool name
- Link (GitHub or primary source URL)
- Approximate star count
- Primary language
- One-sentence factual description
- License (if known)
- Category

Descriptions should be factual and avoid marketing language.

---

欢迎贡献！新增工具时请提供：

- 工具名称
- 链接（GitHub 或主要源地址）
- 大致 star 数量
- 主要语言
- 一句实事求是的描述
- 许可证（如已知）
- 所属分类

描述请保持客观，避免营销话术。
